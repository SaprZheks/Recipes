# Введение
Это моя база рецептов. Она идет как часть [сборника моих заметок](https://github.com/SaprZheks/Study-notes), поэтому если вы уже пользуетесь тем репозиторием, рецепты автоматически встроится в него при запуске ярлыка.
Ниже написаны шаги, которые нужно предпринять, чтобы получить доступ к рецептам.

# Инструкция для общего использования с остальными заметками
Т. к. рецепты - это часть другого репозитория, достаточно открыть ярлык (см. README главного репозитория), что позволит подтянуть все изменения из облака, а затем слева внизу ткнуть на название хранилища с заметками:

  <img src="readme_guide/1_3.png" alt="1" width="400"/>
</div>

Затем нужно нажать на "Управление хранилищами", в появившемся окне нажать на "Открыть папку как хранилище" и выбрать папку `Кухня`.
<div align="center">
  <img src="readme_guide/1_4.png" alt="1" width="400"/>
</div>

**Готово! Рецепты лучше открывать в режиме отдельного хранилища (иначе криво работать будет). Теперь при каждом заходе по старому ярлыку у вас будет доступ к актуальной версии рецептов и заметок!**

# Инструкция для одиночного использования
1. Выберите место, где локально хотите хранить рецепты, откройте в этой папке `PowerShell` и выполните

  `git clone https://github.com/SaprZheks/Recipes.git "Кухня"`

<div align="center">
  <img src="readme_guide/1_2.png" alt="1" width="400"/>
</div>

<div align="center">
  <img src="readme_guide/2_2.png" alt="2" width="400"/>
</div>

2. Затем установите программу [Obsidian](https://obsidian.md/download) для просмотра заметок для всех пользователей и нажмите `Запустить` в конце установки (Вернемся к этому окну позже).
3. Перейдите в репозиторий (папка `Кухня`).
4. В корне репозитория лежит `Obsidian.bat`. Это файл для запуска Obsidian и автоматического подтягивания всех изменений из этого репозитория. На рабочем столе создайте ярлык, указывающий на `Obsidian.bat`, для красоты можете добавить ему иконку из корня репозитория `Obsidian.ico`.

В уже открытом окне Obsidian, выберите "Русский язык" и нажмите "Открыть папку как хранилище", укажите клонированную папку с рецептами (`Кухня`):
